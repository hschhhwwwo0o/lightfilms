## Setup

**To start, you need a [NodeJS]** <br>
**Install && setup project**

```sh
$git clone https://github.com/ssandry/lightfilms.git
$cd lightfilms
$npm install # or yarn
```

## Run

**To run, you need a [NodeJS]**

```sh
$npm run dev # or yarn dev
```
