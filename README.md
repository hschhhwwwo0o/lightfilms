# #Lightfilms

# Dependencies

<li>Node.js</li>
<li>Python3</li>

# Check dependencies

```sh
$python3 -version
$node -v
```

# Setup

**Install && setup project**

```sh
$git clone https://github.com/ssandry/lightfilms.git
$cd lightfilms
$npm install # or yarn
```

# Scripts

**Available scripts for the project**

## build

```sh
$npm run build # or yarn build
```

## start 

```sh
$npm run start # or yarn start
```

## dev

```sh
$npm run dev # or yarn dev
```
